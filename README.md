## CS 340 Portfolio Reflection

### How do you write programs that are maintainable, readable, and adaptable?
I write programs that are maintainable and readable by organizing code into clear sections and using meaningful variable and function names. In this project, I used a CRUD Python module to separate the database logic from the dashboard code. This made the program easier to understand and update without affecting other parts of the system. The advantage of this approach is that the same CRUD module can be reused in other applications that require database interaction, such as future dashboards or backend systems.

### How do you approach a problem as a computer scientist?
I approach problems by breaking them down into smaller steps and solving each part one at a time. For this project, I first focused on connecting to the database, then building the dashboard layout, and finally connecting everything using callbacks and queries. This approach helped ensure that each part worked before combining everything together. Compared to previous assignments, this project required integrating multiple technologies, which made the problem-solving process more structured and iterative.

### What do computer scientists do, and why does it matter?
Computer scientists design and build systems that help organizations store, manage, and analyze data. In this project, the dashboard allows a company like Grazioso Salvare to quickly filter and visualize animal data, which improves efficiency and decision-making. This type of work matters because it helps turn large amounts of data into useful information that can support real-world decisions.
